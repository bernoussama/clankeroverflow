# Code of Conduct

## Our Standard

Be respectful, constructive, and considerate. Harassment, discrimination,
personal attacks, and publishing another person's private information are not
acceptable.

## Reporting

Report unacceptable behavior privately through the repository owner's GitHub
profile. Reports will be reviewed and addressed as promptly and fairly as
possible.
